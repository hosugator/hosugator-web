# Amazon Lightsail

## 요약 (Summary)
가상 서버, 스토리지, 네트워킹 및 데이터베이스를 하나의 고정 가격 플랜으로 제공하는 단순화된 클라우드 플랫폼

## 배경 (Background)
- 탄생 배경: EC2의 복잡한 설정(VPC, 보안 그룹, EBS 등)은 클라우드 입문자나 소규모 웹사이트 운영자에게 높은 진입 장벽이 됨
- 핵심 과제: 세부적인 인프라 제어보다는 '단순한 배포'와 '예측 가능한 비용'이 최우선인 워크로드를 위한 환경이 필요함

## 솔루션 (Solution)
- 핵심 설계: 인스턴스, SSD 기반 스토리지, 데이터 전송, DNS 관리, 정적 IP를 하나의 번들로 묶어 제공
- 작동 메커니즘: 클릭 몇 번으로 워드프레스, 노드JS 등 사전 구성된 블루프린트를 배포하며, 트래픽 폭주 시 EC2로 업그레이드할 수 있는 경로 제공

## 변별점 (Distinction)
- 비유: 
	 - [Lightsail]: 모든 가구가 갖춰진 '풀옵션 원룸' (몸만 들어가면 됨)
	 - [EC2]: 벽지부터 전기 배선까지 직접 설계해야 하는 '신축 빌라' (전문적이지만 복잡함)
- 대안과의 차이: 
	 - [EC2]: 세밀한 성능 튜닝과 확장성이 가능하지만 관리 부하가 높음. Lightsail은 기능은 제한적이나 운영이 압도적으로 쉬움
