# AWS Resource Access Manager (RAM)
## 정의
AWS 계정, 조직(Organization), 조직 단위(OU), 또는 [[IAM]] 역할이나 사용자와 AWS 리소스를 안전하게 공유할 수 있도록 지원하는 서비스
## 원리
### 중앙 집중식 리소스 공유
- 단일 계정에서 생성된 리소스를 다른 계정의 사용자나 역할이 마치 자신의 계정 리소스인 것처럼 접근하고 사용할 수 있도록 리소스 공유(Resource Share)를 생성한다.
- 이 기능을 통해 중앙 네트워크팀이 하나의 [[Transit Gateway]]나 서브넷을 생성하고, 다른 애플리케이션 계정들이 이를 공유하여 사용할 수 있게 된다.
### AWS Organizations 통합
- AWS Organizations와 통합되어 조직 내의 모든 계정 또는 특정 조직 단위(OU)와 리소스를 자동으로 공유할 수 있어 관리 부담이 줄어든다.
## 요소
- 리소스 공유(Resource Share): RAM에서 생성하는 기본 엔티티로, 공유할 리소스(Resource)와 리소스를 사용할 주체(Principal) 목록을 정의한다.
- 리소스(Resource): 공유 대상이 되는 AWS 서비스 리소스이다. (예: [[VPC]] 서브넷, [[Transit Gateway]], [[Route 53]] Resolver 규칙 등)
- 주체(Principal): 공유된 리소스를 사용할 권한을 부여받는 AWS 계정, 조직, OU, [[IAM]] 사용자 또는 역할이다.
- 초대(Invitation): 주체가 AWS Organization의 구성원이 아닐 경우, 리소스를 사용하기 전에 초대를 수락해야 한다.
## 특징
- 다중 계정 관리 단순화: 복잡한 크로스-계정 [[VPC Peering]]이나 개별 리소스 설정 없이도 리소스를 공유할 수 있다.
- 중앙 네트워킹/보안: 중앙 계정에서 Transit Gateway, 서브넷 등 공통 인프라를 프로비저닝하고, 이를 모든 하위 계정과 공유하여 일관된 인프라를 유지할 수 있다.
- 비용 절감: 여러 계정에서 동일한 인프라 리소스(예: Transit Gateway)를 중복 생성할 필요가 없어진다.
- 보안 제어: 공유되는 리소스에 대한 접근 권한은 공유하는 계정의 리소스 기반 정책과 공유받는 계정의 [[IAM]] 정책을 통해 명시적으로 제어된다.
## 비교
| 구분    | AWS RAM (Resource Access Manager)                          | AWS Identity and Access Management (IAM) |
| :---- | :--------------------------------------------------------- | :--------------------------------------- |
| 주요 역할 | 리소스 자체를 다른 계정의 주체와 공유                                      | 리소스에 대한 권한을 계정 내의 주체와 제어                 |
| 초점    | 리소스의 사용 권한을 계정 간에 확장                                       | 사용자 및 역할의 행동 권한을 계정 내에서 정의               |
| 관계    | RAM은 리소스를 공유하고, 공유된 리소스에 대한 실제 접근 제어는 여전히 IAM 정책에 의해 결정된다. | RAM은 IAM의 상위 계층 역할을 수행한다.                |
## 예시
- 중앙 네트워킹 허브 공유:
    1. 중앙 네트워킹 계정에서 AWS TGW를 생성한다.
    2. AWS RAM을 사용하여 이 TGW를 조직 내의 개발, 테스트, 프로덕션 모든 애플리케이션 계정과 공유하는 리소스 공유를 생성한다.
    3. 각 애플리케이션 계정은 공유된 TGW에 자신의 [[VPC]]를 연결하고, 중앙 집중식으로 정의된 라우팅 정책을 통해 다른 계정의 [[VPC]] 및 온프레미스 네트워크와 통신할 수 있다.
    4. 이를 통해 모든 계정은 동일한 고성능 네트워크 허브를 사용하게 되며, 네트워킹 리소스의 중복 생성을 방지한다.
