# Amazon Simple Storage Service (S3)
## 정의
무제한 규모의 객체 스토리지를 인터넷 어디서나 즉시 저장·검색할 수 있는 관리형 스토리지 서비스
## 구성
- [[Bucket]] : 전역 고유 이름을 가진 객체 컨테이너
- [[Object]] : 최대 5 TB 데이터 단위 (키·값·메타데이터·버전 ID)
- [[Storage Class]] : 액세스 빈도에 따른 요금·내구성·가용성 조합
## 특징
- 기준 가용성 99.999999999%(11 9) 내구성
- 버전 관리, 수명 주기, 암호화, ACL, 버킷 정책 지원
- 기본 100버킷/계정 제한(증청 가능)
- 최종 일관성(Overwrite PUT) 또는 강력 일관성(2020-12 이후 새 버킷)
