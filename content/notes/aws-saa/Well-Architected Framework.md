# AWS Well-Architected Framework
## 정의
클라우드 아키텍처의 안정성·보안·성능·비용·운영 우수성을 지속적으로 평가·개선하기 위한 AWS 베스트 프랙티스 모음
## 5대 기둥
- [[Operational Excellence]] : 운영 자동화, 코드 기반 인프라, 지속적 개선
- [[Security]] : ID 관리, 데이터 보호, 네트워크 방어, 감사
- [[Reliability]] : 자동 복구, 다중 AZ, 용량 예측
- [[Performance Efficiency]] : 적절한 리소스 선택, 실험, 모니터링
- [[Cost Optimization]] : 사용량 최적화, 예약 인스턴스, 무상품 제거
## 특징
AWS 공식 검토 도구(Well-Architected Tool)로 워크로드 검토 가능  
각 기둥마다 설계 원칙 5개와 구현 지침 제공
