# Content Delivery Network 
## 정의
지리적으로 분산된 다중 서버 네트워크를 통해 웹 컨텐츠를 빠르고 효율적으로 사용자에게 전달하는 기법
## 원리
- 첫 사용자가 웹사이트에 접속하면 [[CDN]]이 [[DNS]]로 사용자의 위치를 파악하고 [[PoP]] 서버에 캐싱된 콘텐츠를 전달한다.
- 같은 지역의 다음 사용자는 PoP 서버에 이미 캐싱된 콘텐츠가 저장되어 있기 때문에 바로 웹사이트를 이용 가능하다.
- 캐싱된 콘텐츠는 [[TTL]]을 지니며, 사용자 요청 시마다 유효성을 검사하여 콘텐츠를 업데이트한다.
## 특징
- 지연 시간 감소: 사용자 기준 PoP로 캐싱된 콘텐츠를 전달하기에, 지리적 위치와 무관하게 콘텐츠 전달 속도가 보장된다.
- 로드 분산: 다중 서버 네트워크를 활용하기에, 한시적으로 요청이 몰려도 분산 처리로 안정성이 있다.
