# Query
## 정의
데이터베이스 시스템에 특정 요청을 보내는 행위(또는 요청 그 자체)
## 특징
데이터베이스에서 정보를 검색, 추출, 수정, 삭제하거나 정의하기 위해 [[SQL]] 문법에 따라 작성된 명령문이다.
데이터베이스에게 던지는 질문이라고 생각할 수 있다.
예를 들어, "이름이 '김철수'인 모든 고객의 전화번호를 알려줘"라는 질문을 SQL 문법으로 작성한 것이 쿼리이다.
## 예시
- 데이터 검색: `SELECT  FROM Customers WHERE Age > 30;`
- 데이터 삽입: `INSERT INTO Products VALUES ('노트북', 1200000);`
