# Network Hierarchy
## 용어

| 계층 구조                  | 설명                          | 역할                                        |
| -------------------------- | ------------------------------- | --------------------------------------------- |
| 인터넷 (Internet)         | 최상위 계층이자 거대한 연결망            | 전 세계 모든 AS들이 BGP를 통해 상호 연결된 전체 네트워크.          |
| AS (Autonomous System) | 인터넷을 구성하는 독립적인 관리 경계        | 하나의 관리 주체(ISP 또는 대기업)가 내부 라우팅 정책을 결정하는 영역.    |
| 네트워크 (Network)         | AS 내의 논리적인 주소 공간                | IP 주소를 기반으로 분리된 네트워크 세그먼트(예: 192.168.1.0/24). |
| 라우터 (Router)           | 네트워크를 연결하고 경로를 결정하는 하드웨어 장치 | 내부 라우팅 프로토콜(OSPF 등)을 실행하여 패킷의 다음 홉을 결정.       |
## 정의
[[네트워크]]의 일반적인 논리 구성을 계층으로 나타내면 다음과 같다.
`인터넷 > AS > 네트워크 > 라우터`
## 특징
인터넷이라는 전 세계적인 연결망이 있다.
이를 AS들이 독립적인 [[Routing]] 정책을 가지고 구성한다.
각각의 AS는 다중 네트워크로 구성되어 있다.
이 네트워크들을 [[라우터]]가 라우팅 정책에 의거하여 연결한다.
