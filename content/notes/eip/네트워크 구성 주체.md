# 네트워크 구성 주체
## 정의
네트워크의 정상 기능을 위해 필수적인 요소
## 구성 요소
- End device: 사용자가 서비스를 최종적으로 사용하는 장치이며, 정보의 생산/소비를 담당한다. (핸드폰, 컴퓨터, 서버 등)
- Intermediary device: 데이터를 전달하는 장치이며, 출발지에서 목적지까지 경로를 설정하고 관리한다. (라우터, 스위치, 허브, 방화벽 등)
