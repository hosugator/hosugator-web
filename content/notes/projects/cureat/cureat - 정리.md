AI 기반 자연어 분석 및 추천 시스템 구축
기능: 사용자의 자연어 검색어(예: "데이트하기 좋은 분위기 좋은 레스토랑")를 NLP 모델로 분석하고 임베딩합니다. 수집된 맛집 리뷰 또한 임베딩하여 사용자 입력과 유사도를 분석, 개인화된 추천 리스트를 제공합니다.
기술: NLP 모델(형태소 분석, 불용어 처리), 임베딩, 코사인 유사도 알고리즘을 활용했습니다.
효율적인 데이터 수집 및 관리 시스템 개발
기능: 외부 API 및 웹 크롤링을 통해 실시간으로 맛집 정보와 리뷰를 수집하고, 수집된 후기 중 광고성 콘텐츠를 필터링하는 로직을 적용했습니다.
기술: RESTful API 통신, 웹 크롤링 기술을 활용했으며, 수집된 데이터를 임베딩하여 Vector DB에 저장했습니다.
성능 최적화 및 사용자 경험 개선
기능: 사용자의 반복적인 검색에 대한 응답 속도를 높이기 위해 캐싱 시스템을 도입했습니다.
기술: 검색어를 캐시에 저장하여 불필요한 API 호출 및 크롤링을 줄이는 로직을 구현했습니다.
체계적인 프로젝트 관리
도구: Jira를 사용하여 프로젝트의 에픽, 스토리, 태스크를 정의하고 진행 상황을 관리했습니다.
결과: 효율적인 업무 분배 및 협업을 통해 프로젝트 목표를 달성했습니다.
