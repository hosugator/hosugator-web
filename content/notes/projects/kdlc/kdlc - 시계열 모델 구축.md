# 시계열 모델 테스트 전략

## 핵심 용어 정의
### 시계열 모델 핵심 용어 정의표
| 용어 | 정의 |
| :--- | :--- |
| SARIMA | 계절성을 포함하는 시계열 데이터의 추세와 주기를 모델링하는 데 사용되는 전통적인 통계 모델이다. |
| LSTM | 순환 신경망(RNN)의 한 종류로, 시퀀스 데이터 내의 장기 의존성(Long-term Dependency)을 학습하는 데 특화된 딥러닝 모델이다. |
| 시퀀스 길이 | LSTM 모델이 미래를 예측하기 위해 입력으로 사용하는 과거 데이터의 기간이다. (예: 과거 30일) |
| ACF/PACF | 시계열 데이터의 자기상관 및 부분 자기상관을 측정하는 함수이며, SARIMA 모델의 파라미터(p, q)를 결정하는 데 활용된다. |
| 앙상블 (Simple Voting) | 여러 개의 독립적인 모델(시계열 + ML)의 예측 결과를 단순 평균하여 최종 예측치를 도출하는 방식이다. |
| 비정상성 | 시계열 데이터의 평균이나 분산이 시간에 따라 변하는 성질이며, SARIMA 모델은 차분(d)을 통해 이를 해소한다. |

## 1. SARIMA 모델 구축 전략
SARIMA(Seasonal AutoRegressive Integrated Moving Average) 모델은 데이터의 추세(Trend)와 계절성(Seasonality)을 명시적으로 모델링하는 데 활용된다.

* 모델링 목적: 데이터에 내재된 장단기 주기성과 비정상성(Non-stationarity)을 파악하여 예측에 반영하는 것이다.
* 파라미터 설정: ACF(자기상관 함수)와 PACF(부분 자기상관 함수) 플롯을 분석하여 최적의 파라미터 p, d, q 값을 탐색한다.
* 계절성 주기 설정: 물류 수요 데이터의 특성을 반영하여 두 가지 주요 계절성 주기를 설정한다.
    * 주간 계절성: 주기 S=7일 (요일 효과 반영)
    * 월간 계절성: 주기 S=30일 (월별 및 월초/월말 효과 반영)

## 2. LSTM 모델 구축 전략
LSTM(Long Short-Term Memory)은 순환 신경망(RNN)의 한 종류로, 복잡하고 비선형적인 시계열 패턴을 포착하는 데 사용되는 딥러닝 기반 모델이다.

* 모델링 목적: 과거 시퀀스를 입력받아 미래를 예측하는 딥러닝 기반 시퀀스 모델을 구축하는 것이다.
* 시퀀스 길이 설정: 과거 데이터 중 얼마만큼의 기간을 참고하여 미래 시점을 예측할지 결정하는 시퀀스 길이를 설정한다. 예를 들어, 과거 30일의 수량 및 특징 데이터를 입력으로 사용하여 다음 날의 수량을 예측하도록 설정한다.
* 하이퍼파라미터 설정: LSTM 레이어 구성, 은닉층 크기, 드롭아웃(Dropout) 비율, 배치 크기(Batch Size), 학습 횟수(Epoch) 등을 최적화하여 모델을 훈련한다.

## 3. 최종 예측 모델 및 앙상블 전략
각 카테고리별 매입/매출 예측은 총 8개의 핵심 예측 과제로 구분된다. 예측의 안정성과 정확도를 극대화하기 위해, 시계열 모델과 머신러닝(ML) 모델을 결합한 단순 투표(Simple Voting) 앙상블 모델을 최종 예측기로 사용한다.

* 앙상블 모델 구성: 각 핵심 과제마다 1개의 시계열 모델(SARIMA/LSTM 중 선택)과 1개의 ML 모델(LightGBM, XGBoost, Random Forest 중 선택)이 결합된다. 
* 최종 예측은 두 모델의 결과를 결합하는 단순 투표 방식으로 도출된다.

| 물류센터 | 카테고리 | 거래 유형 | 모델 담당자 | 예측 모델 구성 (총 16개 요소) |
| :--- | :--- | :--- | :--- | :--- |
| A | 생수/음료/건강 | 매입 | 하림 | 시계열 모델 + ML 모델 |
| A | 생수/음료/건강 | 매출 | 하림 | 시계열 모델 + ML 모델 |
| A | 신선식품 | 매입 | 성봉 | 시계열 모델 + ML 모델 |
| A | 신선식품 | 매출 | 성봉 | 시계열 모델 + ML 모델 |
| B | 탄산음료 | 매입 | 대희 | 시계열 모델 + ML 모델 |
| B | 탄산음료 | 매출 | 대희 | 시계열 모델 + ML 모델 |
| B | 봉지라면 | 매입 | 예희 | 시계열 모델 + ML 모델 |
| B | 봉지라면 | 매출 | 예희 | 시계열 모델 + ML 모델 |
