# 수식 필드 사용 (Use Formula Fields)
## 수식 필드 소개
**Formula Fields**는 레코드의 데이터를 기반으로 **자동으로 값을 계산하고 표시**하는 강력한 사용자 정의 필드입니다. 
사용자가 별도의 계산 없이 핵심 데이터를 한눈에 이해할 수 있도록 돕습니다.
* **용도:** 두 숫자 필드를 나누어 백분율 생성, 필드를 클릭 가능한 하이퍼링크로 전환, 두 날짜 사이의 일수 계산 등 다양한 작업이 가능합니다.
* **이점:** 계산된 값을 페이지 레이아웃, 보고서, 목록 보기에 추가하여 사용자들이 **핵심 정보**에 즉시 접근할 수 있게 합니다.
## 수식 편집기 (Formula Editor)
수식 필드는 모든 표준 또는 사용자 정의 객체에서 생성할 수 있습니다. 수식 편집기는 수식을 작성하고 디버깅하는 데 사용됩니다.
### 편집기 구성 요소
| 기능 | 설명 |
| --- | --- |
| **Simple / Advanced** | **Advanced** 편집기 사용을 권장합니다. 더 많은 도구를 제공합니다. |
| **Insert Field** | 수식에 사용할 필드를 선택하며, 올바른 **구문(Syntax)**을 자동으로 생성합니다. |
| **Insert Operator** | 사용 가능한 수학 및 논리 연산자(+, -, *, /, AND, OR 등)를 삽입합니다. |
| **Functions** | Salesforce에서 미리 구현된 복잡한 연산입니다. (예: **TODAY()**는 현재 날짜를 반환, **LEN(text)**는 텍스트 길이를 찾음) |
| **Check Syntax** | 수식을 저장하기 전에 구문 오류가 없는지 확인하는 데 필수적인 도구입니다. |
### 수식 작성 규칙
* **공백:** 공백(Whitespace)과 줄 바꿈은 수식 실행에 영향을 주지 않습니다.
* **대소문자:** 필드 및 객체 이름의 **대소문자**를 구분합니다.
* **연산 순서:** 숫자를 다룰 때는 표준 **연산 순서**가 적용됩니다.
## 수식 필드 예제
### 1. Cross-Object Formula (교차 객체 수식)
하나의 객체(예: Contact)에서 관련 객체(예: Account)의 필드를 참조하여 표시하는 수식입니다.

| 객체 | 필드 레이블 | 반환 유형 | 수식 |
| :--- | :--- | :--- | :--- |
| **Contact** | Account Number | Text | **Account.AccountNumber** |
### 2. 날짜 계산 (Days to Close)
기회 마감일까지 남은 일수를 계산하여 숫자로 반환하는 수식입니다.

| 객체 | 필드 레이블 | 반환 유형 | 수식 |
| :--- | :--- | :--- | :--- |
| **Opportunity** | Days to Close | Number | **CloseDate - TODAY()** |
### 3. 하이퍼링크 및 할인 계산
* **하이퍼링크:** **HYPERLINK()** 함수를 사용하여 외부 웹사이트로 연결되는 클릭 가능한 하이퍼링크를 생성합니다.
* **할인 계산:** **ROUND()** 함수를 사용하여 기회 금액에 할인을 적용하고 결과를 반올림할 수 있습니다.
    * *예시 수식:* `ROUND(Amount * 0.88, 2)` (12% 할인 후 소수점 둘째 자리까지 반올림)
### 4. 논리적 체크박스 (Checkbox Formula)
특정 조건이 참(True)일 때 체크 표시가 나타나도록 하는 수식입니다.
* *예시:* 기회 계정의 직원 수가 1,000명 초과 **AND** 기회 금액이 $10,000 초과일 때 체크 표시.
* *사용 함수:* **AND**
## 수식 디버깅 및 일반적인 오류
**Check Syntax** 버튼은 디버깅에 필수적입니다. 자주 발생하는 구문 오류는 다음과 같습니다.

| 오류 유형 | 원인 | 디버깅 팁 |
| :--- | :--- | :--- |
| **Missing parentheses** | 여는 괄호와 닫는 괄호의 수가 일치하지 않거나, 함수 매개변수 사이에 쉼표를 누락했을 경우입니다. | 여러 줄로 나누어 작성하거나, 쉼표를 확인합니다. |
| **Incorrect parameter type** | 함수가 예상하는 데이터 유형(예: Text) 대신 다른 유형(예: Number)을 입력한 경우입니다. | 함수의 예상 매개변수 유형을 확인합니다. |
| **Incorrect number of parameters** | 함수에 필요한 매개변수보다 많거나 적게 입력한 경우입니다. | 함수의 문서나 도움말을 참고하여 매개변수 개수를 확인합니다. |
| **Formula result is incompatible** | 수식 필드 생성 시 선택한 반환 유형과 수식이 실제로 반환하는 데이터 유형이 다른 경우입니다. | 수식의 결과를 예상되는 반환 유형(예: Number)과 일치시켜야 합니다. |
| **Field does not exist** | 필드 이름의 철자가 틀렸거나, 대소문자가 잘못되었거나, 텍스트 리터럴에 **인용 부호(`""`)**를 누락한 경우입니다. | **Insert Field** 메뉴를 사용하여 올바른 구문을 삽입하는 것이 가장 안전합니다. |
| **Unknown function** | Salesforce에서 지원되지 않는 함수를 사용했거나 함수 이름을 잘못 입력한 경우입니다. | 함수 이름의 철자를 확인하고, Salesforce 함수 목록에서 지원 여부를 확인합니다. |
