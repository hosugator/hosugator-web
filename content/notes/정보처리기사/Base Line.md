# Base Line
## 정의
공식 승인되어 고정된 소프트웨어 형상 항목들의 집합
## 특징
개발 과정에서의 기준점이자, 형상 통제의 시작점이다.
소프트웨어의 특정 버전을 명확하게 정의하여, 소프트웨어의 무결성을 보장한다.
## 종류
| 베이스라인 종류 | SDLC 단계    | 고정되는 형상 항목              |
| ------------ | -------------- | --------------------------- |
| 기능 베이스라인 | 요구사항 분석 완료     | 요구사항 명세서, 시스템 개요 등          |
| 할당 베이스라인 | 설계 완료          | 시스템 설계 명세서, 인터페이스 정의서 등     |
| 제품 베이스라인 | 최종 테스트 완료 / 배포 | 최종 소스 코드, 사용자 매뉴얼, 운영 설명서 등 |
