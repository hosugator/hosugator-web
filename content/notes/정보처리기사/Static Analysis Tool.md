# 정적 분석 도구
## 정의
소스 코드 텍스트만으로 코딩 오류, 잠재적인 보안 취약점, 유지보수성을 저해하는 스타일 문제 등을 자동적으로 점검하여, 실행 전 문제를 찾아내어 해결 비용 절감
## 원리
코딩 표준 및 스타일 준수: 변수명 규칙, 들여쓰기, 주석 사용 등 일관성 점검을 수행한다.
코드 복잡도 측정: 순환 복잡도(Cyclomatic Complexity)와 같은 지표를 계산하여 유지보수가 어려운 코드를 식별한다.
잠재적 결함 발견: 초기화되지 않은 변수 사용, 도달 불가능한 코드(Dead Code), 메모리 누수 가능성 등을 탐지한다.
보안 약점 분석: SQL 인젝션, XSS(Cross-Site Scripting) 등 보안 취약점으로 이어질 수 있는 코드를 탐지한다.
